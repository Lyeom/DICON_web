PACIFIC

* 모든 파일 안에는 이미지 확인용의 jpg와 원본 psd가 있습니다. 제가 죽었으면 이거보고 참고하시면 됩니다.
* 모든 페이지의 흰색 텍스트 #f2f2f2 이고 검은 텍스트는 #5c5c5c 입니다.
* 모든 페이지의 배경은 똑같습니다.(재량껏 우려먹으시면됩니다)
* 친구추가 아이콘과 undo아이콘, 프로필사진의 틀(원형)도 다 똑같습니다.(이것도 재량껏 우려드시면됩니다)


========================================================================================================
login_page - 맨 첫페이지입니다. 로그인을 합시다

* title - 타이틀 로고입니다.
* login_bar - 이메일과 비밀번호를 입력하는 바(bar)입니다.
* login_button - 정보를 입력한후 로그인할 때의 버튼입니다.
* signup_bar - 회원가입을 할 때 누르는 버튼입니다.
* facebook_bar - 페이스북연동 로그인 버튼입니다.
* login_bg - 맨 첫페이지의 뒷배경입니다.

========================================================================================================
signup_page - 회원가입 페이지입니다.

* signup_title - 회원가입 페이지의 타이틀 로고입니다.
* login_data - 회원가입할 때의 정보를 입력할 수 있는 창입니다.
* ok_button - 정보입력 후 완료버튼입니다
* signup_bg - 회원가입 페이지의 뒷배경입니다.

========================================================================================================
matching_page - 매칭을 찾을 때의 페이지입니다(이미지 확인용 폴더에 페이지 jpg2개가 다르니 확인해주세요)

* add_friend_icon - 친구추가시 누르는 버튼입니다
* bar - 리소스와 이미지 확인용 폴더의 jpg를 확인해주세요
* check_icon - 친구추가시 수락할 때의 버튼입니다.
* freind_icon - 친구창을 뜨게하기위한 아이콘입니다.(누르면 친구창이 뜹니다)
* friendlist - 친구목록입니다.
* matching_bg - 이 페이지의 뒷배경입니다.
* money_bar - 현재 소유하고있는 진주(돈)이 얼마나 있는지 알려주는 바(bar)입니다.
* player_bar - 내정보가 나타나는 바(bar)입니다.
* profile_circle - 친구창에 들어갈 작은 프로필 사진틀입니다.
* rank_icon - 랭킹 페이지로 갈수있는 아이콘입니다.
* search_icon - 이메일로 친구를 찾을 때 쓰는 바(bar)입니다.
* refusal_icon - 친구를 받고싶지않을 때 누르는 아이콘입니다.
* shop_icon - 상점으로 갈 수 있는 아이콘 입니다.
* vs_icon - 친구와 결투를 하고싶을때 누르는 아이콘입니다.

========================================================================================================
Rank page - 랭킹 페이지 입니다

* add_friend_icon - 랭킹바에 들어갈 친구추가 버튼입니다. 마찬가지로 친구로 추가하고싶을 때 쓰입니다.
* profile_circle - 위(matching_page)페이지와 쓰임이 같습니다.
* rank_bar - 회원의 정보가 들어갈 바(bar)입니다.
* rank_bg - 랭킹페이지의 뒷배경입니다.
* rank_title - 랭킹페이지의 타이틀 로고 입니다.
* title_small - 페이지 우측 상단에 들어갈 작은 타이틀입니다.
* undo_button - 뒤로가기 버튼입니다.

========================================================================================================
Shop_page - 상점 페이지 입니다

* card_frame - 이모티콘카드가 들어갈 틀입니다. 이 안에 카드를 넣어주시면 됩니다.(카드는 추후에 디자인 할 예정입니다)
* category_frame - 이미지 확인용 폴더를 확인해주세요.
* lock_icon - 스킨옆에 들어갈 잠금 아이콘입니다.
* money_bar - 현재 소유한 진주의 량을 나타내주는 바(bar)입니다. 
* next_button - 옆으로 넘길 때 누르는 버튼입니다.
* shop_bg - 상점페이지의 뒷배경입니다.
* title_small - 랭킹페이지와 같습니다.
* undo_button - 뒤로가기 버튼 입니다.


========================================================================================================
ingame_page - 인게임 페이지입니다.

* angry_icon - 누르면 화난 이모티콘 카드가 나오는 아이콘입니다. 
* dead_icon - 누르면 넌 이미 죽어있다 같은 카드가 나오는 아이콘입니다.
* smile_icon - 누르면 웃는 이모티콘 카드가 나오는 아이콘 입니다.
* inoticon_bar - 이모티콘 박스입니다.(이미지 확인용 폴더참고 필수)
* player_bar - 플레이어의 정보를 알려주는 바(bar)입니다.
